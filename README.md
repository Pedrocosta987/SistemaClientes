# SistemaClientes

Um projeto simples desenvolvido em Java utilizando Programação Orientada a Objetos (POO).  

Funcionalidades

- Cadastro de clientes  
- Listagem dos clientes cadastrados  
- Remoção de clientes  
- Busca por nome  
- Estrutura totalmente baseada em POO

---

Tecnologias utilizadas

- Java (JDK 17+)  
- IntelliJ IDEA  
- Git + GitHub

---

Conceitos de POO aplicados

- Classes e Objetos  
- Encapsulamento  
- Construtores  
- Métodos Get/Set  
- ArrayList para armazenamento

---

src/
├── Cliente.java
├── GerenciadorClientes.java
